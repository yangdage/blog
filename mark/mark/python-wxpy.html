python脚本给微信发消息

Notes
2018-02-16
mark

最近有个小需求，需要每天查询前一天的经营统计数据给到合作方。开始几天还算积极。渐渐的，有些天就忘记了，特别是节假日。后来就想到要找一个能每天定时自动查询和发送这些消息给已经建好的微信群里就省的忘记了。

花了点时间研究了下python的两个相关项目itchat和wxpy。

由于wxpy是基于itchat的修改和再封装，使用起来更方便，文档也比较清晰简洁，遂最后选择了wxpy来实现相关功能。

wxpy 在 itchat 的基础上，通过大量接口优化提升了模块的易用性，并进行丰富的功能扩展，自称可能是最优雅的微信个人号API

wxpy能干什么

wxpy可用来实现各种微信个人号的自动化操作，具体如下

  • 控制路由器、智能家居等具有开放接口的玩意儿
  • 运行脚本时自动把日志发送到你的微信
  • 加群主为好友，自动拉进群中
  • 跨号或跨群转发消息
  • 自动陪人聊天
  • 逗人玩
  • …

安装wxpy

直接执行下面命令即可

pip install wxpy
